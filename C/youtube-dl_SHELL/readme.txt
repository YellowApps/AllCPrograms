youtube-dl SHELL

Как пользоваться?

1) Откройте нужное видео на YouTube.
2) Нажмите Поделиться -> Скопировать ссылку.
3) Запустите shell.exe из этой папки
4) Вставьте ссылку (ПКМ / Ctrl + V) и нажмите Enter.
5) Дождитесь окончания загрузки.
6) Видео будет сохранено в эту папку.

Что делать, если возникли ошибки?

Посмотрите на ошибку. Если это:

текст в основном окне youtube-dl SHELL, запустите update.bat и повторите загрузку;

ошибка в диалоговом окне - запустите install.bat от имени администратора (если появится ещё одна ошибка - не страшно), и нажмите Enter для перезагрузки ПК.
